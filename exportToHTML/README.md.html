1    # The Cat Shelter
2    
3    _An example that, for some reason, spawned a life of its own._
4    
5    _Before starting, Yes, it is a real cat. Yes, it has a shelter. Yes,
6    the shelter has a camera and security devices. Evidence is in the `img`
7    folder._
8    
9    ## The Problem
10   
11   A kindly cat owner has installed a cat shelter in his garden. This acts 
12   as a refuge for his feline friend should the creature be caught outside in 
13   inclement weather at a time when its human servant is unavailable to open the 
14   back door.
15   
16   The cat shelter contains a small camera, along with a motion detector and an 
17   RFID reader that detects a chip in a tag attached to the cat's collar. This 
18   is connected via WiFi to a laptop in the house, where a program creates a log of 
19   when the cat enters and leaves the shelter. If the chip is not detected, but 
20   motion is, it can be assumed that the neighbour's cat has trespassed into the 
21   shelter. In this case, a loud siren plays automatically, and jets of ice cold 
22   water are directed at the intruder. The ferocity of the attack is such that 
23   the cat can be assumed to have stayed for just one minute.
24   
25   There has been a significant investment in this project, and the cat owner 
26   is keen to find out how much the shelter is being used. A program to analyse a 
27   daily log file is required.
28   
29   ## Detailed Specification
30   
31   The laptop stores a data stream in a file that can be analysed. It records when a 
32   cat entered, when it left, and whether the cat was an intruder. One line is written 
33   for every cat activity, so a single line contains both the entry and departure times.
34   
35   Times are stored as minutes, with a start of midnight. So midnight would be 
36   represented as zero, 1am is 60, and 9:30am would be 570 (9 x 60 + 30). There 
37   is one file created each day.
38   
39   The cat shelter is small, so there can only be one cat in it at a time. It is 
40   often empty as the cat likes to bask on the roof.
41   
42   The format of the file is shown by the following short sample.
43   
44   ```text
45   OURS,600,630
46   THEIRS,700,701
47   OURS,842,900
48   THEIRS,1000,1001
49   THEIRS,1010,1011
50   END
51   ```
52   
53   So we see that the correct cat (``OURS``) entered the shelter at 
54   10:00am (time 600), and stayed for 30 minutes (630 - 600). A 
55   different cat entered at time 700 which is 11:40, and swiftly left, having
56   been attacked with sound and water. And so on.
57   
58   The data stream ends ``END``, which is written automatically 
59   before the file closes at 23:59. The shelter is never occupied at this time
60   (the cat being indoors in its basket). 
61   
62   It is safe to assume that the file is always in this format.
63   
64   The required output from the analysis program will be:
65   
66   * The total number of times the correct cat has entered the house.
67   * The number of times intruding cats have been doused with water.
68   * The total time spent in the house by the correct cat.
69   * For the correct cat only, the average duration of each visit, the 
70   duration of the longest visit, and the duration of the shortest visit.
71   
72   The name of the data file should be provided as a command-line 
73   argument. The program should therefore run from the command-line. It should 
74   handle the common errors associated with files on the command-line, such as
75   missing arguments, files that cannot be opened, and so on.
76
